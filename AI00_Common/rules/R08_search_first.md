# R08 — 新项目先搜索规则

> 适用：AI00框架下所有项目

---

## 触发条件

用户说「**建项目**」或「**帮我做 [新功能]**」，且满足以下任一条件：
- 预计超过 2 小时工作量
- 涉及新技术/新领域（不是已做过的类型）
- 用户主动说「先搜搜看」

## AI 执行步骤

1. **WebSearch** — 搜索「[项目类型] github [关键词] 2026」
2. **整理对比表** — 列出找到的前 3-5 个方案：

| 项目 | Stars | 优点 | 缺点 | 和我们的差异 |
|------|-------|------|------|------------|
| xxx | ⭐100 | ... | ... | ... |

3. **给出建议** — 推荐「从零做」还是「借鉴 XX」还是「直接用 XX」
4. **等用户确认** — 用户说「开始做」后才动手

## 不触发的情况

- 小改动（修 bug、加字段、改样式）
- 用户明确说「直接做」「不用搜索」
- 同类型项目已做过（如第二个 React Native app）

## 搜索关键词模板

```
[功能描述] github claude code 2026
[技术栈] [功能] open source
awesome-claude-code [关键词]
```

# H31 — Obsidian 手机设置指南

> 目的：在 iPhone / Android 上完整查看 T00 的 .md 文件（含表格、代码块、标题层级）

---

## 什么是 Obsidian

专门处理 Markdown 文件的笔记 App。免费，离线可用，支持 .md 格式，渲染效果和 VS Code 预览完全一样。

---

## iPhone 安装步骤

1. App Store 搜索 **「Obsidian」** → 安装（免费）
2. 打开 App → 「Create new vault」→ 起名「T00」
3. 位置选 **iCloud Drive**（这样电脑也可以访问同一个 vault）

---

## Android 安装步骤

1. Play Store 搜索 **「Obsidian」** → 安装（免费）
   （国内设备：直接从 obsidian.md 官网下载 APK）
2. 打开 App → 「Create new vault」→ 选择手机本地文件夹

---

## 把 T00 文件同步到 Obsidian（两种方法）

### 方法A：iCloud（iPhone 推荐）

1. 在电脑上安装 iCloud for Windows
2. 把 T00 文件夹复制到 `C:\Users\user\iCloudDrive\Obsidian\T00\`
3. 等同步完成 → iPhone Obsidian 自动出现文件

### 方法B：OneDrive + 手动同步

1. 把 T00 文件夹复制到 OneDrive
2. 手机 OneDrive App 下载文件夹
3. Obsidian 打开该本地文件夹

### 方法C：直接传文件（最简单，适合偶尔查看）

1. 电脑 → WeChat/邮件 → 发送单个 .md 文件给自己
2. 手机收到 → 「打开方式」→ 选 Obsidian
3. 自动导入到 vault

## 使用技巧

| 操作 | 方法 |
|------|------|
| 搜索文件 | 左上角搜索图标 |
| 切换预览/编辑 | 右上角铅笔图标 |
| 查看表格/代码块 | 默认渲染，直接显示 |
| 关键词搜索 | 放大镜 → 输入关键词 |

## 和 OneDrive 的区别

| | OneDrive | Obsidian |
|--|---------|---------|
| 格式渲染 | 纯文字 | ✅ 完整 Markdown |
| 离线使用 | ❌ 需要网络 | ✅ 本地 |
| 搜索 | 基础 | ✅ 全文搜索 |
| 推荐场景 | 临时查看 | ✅ 日常使用 |

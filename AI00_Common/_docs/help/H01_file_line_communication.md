## 两个专用沟通文件

| 文件 | 位置 | 用途 |
|------|------|------|
| `Req1.md` | `F:\T00\Req1.md` | 写好了，告诉 AI 读 |
| `Req2.md` | `F:\T00\Req2.md` | 草稿，还在写，AI 不用管 |

---

## 使用方法

### 基础写法
```
Req1.md 第20行后更新了，请看
```

### 指定范围（更精确，更省token）
```
Req1.md 20-35行，新增了命名规则，请看
```

### 多处更新
```
Req1.md 第20行、第45行都更新了，请看
```

---

## 什么时候用

| 情况 | 建议 |
|------|------|
| 内容超过3-4行 | 写 Req1.md + 告诉行号 ✅ |
| 短句子、简单指令 | 直接在聊天框说 ✅ |
| 需要 AI 看整个文件 | 直接说「请读 Req1.md」 |

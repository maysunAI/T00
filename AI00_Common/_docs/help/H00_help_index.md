# H00 — 帮助文档索引

> 你是否遇到过：和 AI 聊了几十轮，回头想找某个结论却完全找不到？

**AI00 用最简单的方式解决这个问题：**
- 需求写 req，回答写 reply，每条都标行号
- 会话结束自动存档，下次接着来
- 10 分钟上手，今天就能用

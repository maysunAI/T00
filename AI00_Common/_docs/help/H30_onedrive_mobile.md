# H30 — OneDrive 手机查看 T00 文件

> 目的：在手机上随时查看 T00 的 .md 文件

---

## 什么是 OneDrive

微软的云存储服务。你的电脑已经在用（截图路径 `C:\Users\user\OneDrive\`），文件会自动同步到云端。

---

## iPhone 设置步骤

1. App Store 搜索 **「OneDrive」**（微软官方）→ 安装
2. 用微软账号登录（和电脑上同一个账号）
3. 打开 App → 左侧菜单「文件」→ 找到 T00 相关文件夹
4. 点 .md 文件 → 右上角「···」→「打开方式」→ 选文本查看器

> ⚠️ OneDrive 只能显示纯文字，不渲染表格/代码块格式。
> 想要好的 Markdown 渲染效果 → 用 Obsidian（见 H31）

---

## Android 设置步骤

1. Play Store 搜索 **「Microsoft OneDrive」** → 安装
   （或直接用浏览器打开 onedrive.com）
2. 用微软账号登录
3. 同上操作查看文件

---

## 把 T00 文件放进 OneDrive

**方法**：把需要手机看的文件夹复制到 OneDrive 文件夹里

```
# 例：把 RESUME.md 放进 OneDrive
复制 F:\T00\RESUME.md → C:\Users\user\OneDrive\T00\RESUME.md
```

等几分钟自动同步 → 手机 App 里就能看到。

---

## 适合用 OneDrive 的场景

- 临时查看某个文件
- 不想装新 App
- 文件格式要求不高（纯文字即可）

长期、频繁查看 → 推荐 Obsidian（H31）

## 记忆算法对比

| 方法 | 相比艾宾浩斯 | 一句话 |
|------|-----------|--------|
| 艾宾浩斯曲线 | 基准 | 固定时间间隔复习 |
| SM-2（Anki用的）| ⬆️ 更准 | 根据你的记忆成绩调间隔 |
| **FSRS（2023）** | ⬆️⬆️ 目前最优 | 机器学习优化，误差最小 |
| 主动回忆 | 补充技术 | 不看答案，强迫回想→记得更牢 |
| 费曼技巧 | 检验工具 | 用自己的话解释→发现是否真懂 |

**推荐组合：FSRS + 主动回忆 + 费曼自检**

→ PJ16_knowcard 技术路线采用 FSRS 算法。

## AI主动发现知识盲点（D05）

**痛点**：「你不知道你不知道什么」

AI 通过对话记录，主动检测你可能存在的知识盲区，并推送针对性复习。

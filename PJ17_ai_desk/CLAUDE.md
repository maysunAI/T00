# PJ17_ai_desk — AI 工作台（VSCode 风格）

> 核心理念：把 AI00 的日常工作流（req/reply/chat）变成一个像 VSCode 一样的桌面界面。
> 参考：VSCode 布局，Electron 实现。

---

## 界面布局设计

```
┌────────────┬──────────────────────────────────┬────────────┐
│            │  [Tab1][Tab2][Tab3+]             │            │
│  左侧导航   │──────────────────────────────────│  右侧导航   │
│  (可关闭)  │  【Req 区】    │  【Reply 区】    │  (可关闭)  │
│            │  固定左侧      │  固定右侧        │            │
│  Explorer  │                │                 │            │
│  ─────────  │                │                 │            │
│  Projects  │                │                 │            │
│  Tasks     │                │                 │            │
│  讨论话题  │──────────────────────────────────│            │
│  知识库    │  【聊天区（底部）】               │            │
└────────────┴──────────────────────────────────┴────────────┘
```

---

## 各区域功能说明

### 左侧导航（可关闭，分上下两部分）

**上部：Explorer（快速导航）**
- Projects 列表（点击直接切换项目）
- Tasks（按状态分组：进行中 / 待做 / 完成）
- 讨论话题（D01-D0x，按项目分组）
- 知识库（PJ16 保存的知识卡片，快速查）

**下部：（待补充）**
- 可能：搜索、设置、会话历史

### 中间区域

**顶部 Tab 栏**
- Tab 可添加、可关闭、可固定（pin）
- 默认固定 Tab：req / reply / chat

**中间左：Req 区（固定）**
- 显示 temp01_req.md 内容
- 可直接编辑

**中间右：Reply 区（固定）**
- 显示 temp03_reply.md 内容
- 只读（AI写，人查看）

**底部：聊天区**
- 输入框 → AI 回复
- 聊天过程中 AI 自动识别并保存：

| 类型 | 触发词/规则 | 保存到 |
|------|-----------|--------|
| 知识 | 用户说「记下来」/ AI 检测到新知识 | PJ16 知识库 |
| Project | 用户说「建项目」| PROJECTS_INDEX |
| Task | 用户说「任务：...」| TASK_INDEX |
| 讨论话题 | 用户说「讨论话题：」| 对应项目 CLAUDE.md |
| 其他 | ？ | 待定（见D01）|

### 右侧导航（可关闭）
- 待定：可能放「当前项目详情」/ 「RESUME 快览」

## 技术选型

| 层 | 技术 | 理由 |
|----|------|------|
| 框架 | Electron | 跨平台桌面，PJ13 已有基础 |
| UI | React + CSS Grid/Flex | VSCode 风格布局 |
| 编辑器 | Monaco Editor | VSCode 同款，支持语法高亮 |
| 文件读写 | Node.js fs | 直接读写 temp0*.md |
| AI 接入 | Claude API | 聊天 + 自动分类收集 |

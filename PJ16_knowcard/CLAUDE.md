# PJ16_knowcard — 知识卡片·AI记忆系统

> 核心痛点：和 AI 聊天学到的东西无法保存→整理→记忆，最后放弃学习。
> 目标：一个 AI 管理的知识卡片系统，手机可访问，科学记忆曲线驱动复习。

---

## 三大核心功能

| 功能 | 描述 |
|------|------|
| 保存 | 聊天中遇到不会的 → 一键保存到系统 |
| 整理 | AI 去重、分类、标优先级、识别知识关联 |
| 记忆 | 艾宾浩斯 + 现代 SRS，最终目标：灰色记忆（永久记忆） |

## 记忆科学参考（AI 建议）

| 方法 | 说明 | 相比艾宾浩斯 |
|------|------|-----------|
| 艾宾浩斯曲线 | 固定时间间隔复习 | 经典，但不够个性化 |
| SM-2 / Anki | 根据回答质量调整间隔 | ⬆️ 更精准 |
| FSRS（2023） | 机器学习优化间隔，最新研究 | ⬆️⬆️ 目前最优 |
| 主动回忆 | 不看答案，强迫自己想 | 提升记忆质量的关键 |
| 费曼技巧 | 用自己的话解释给别人 | 检验"真正理解"还是"表面理解" |

> 推荐：FSRS + 主动回忆 + 费曼自检 = 最佳组合

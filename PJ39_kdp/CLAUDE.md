# PJ39_kdp

> 立项时间：2026-06-11
> 状态：立项中（分析阶段）

## 项目定位

用 T00 + Claude 生成电子书/虚拟产品，在 Amazon KDP（Kindle Direct Publishing）销售。
目标市场：日本（优先）、美国。

## 核心思路

T00 的优势 = AI协作效率高 → 可以快速生成高质量的 how-to 内容 → 出版成 KDP 电子书

## 产品候选（优先级排序）

| 优先级 | 产品 | 目标市场 | 定价 |
|--------|------|---------|------|
| 🥇 1 | Claude Code 活用術（日文 how-to 书）| 日本 | $4.99 |
| 🥈 2 | The Claude Code Playbook（英文 productivity 书）| 美国/英语 | $7.99 |
| 🥉 3 | AI Prompt Collection for Developers（英文 prompt 集）| 美国 | $2.99 |
| 4 | AI生産性向上 実践ガイド（日文 AI工具指南）| 日本 | $5.99 |

## 变现分析

- KDP 版税：70%（$2.99-$9.99 区间）
- 目标：每本书每月 $50-200 被动收入
- 优势：一次写好，永久销售；T00 可以快速迭代更新版本
- 日本市场竞争更小，Claude Code 相关内容几乎空白

## 制作流程

1. 用 Claude 生成初稿（1-2天）
2. 用 T00 框架组织内容、添加截图
3. 用 KDP 发布（格式：.epub 或 Word）
4. 定期更新版本（Claude Code 更新时）
